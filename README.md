# aalukehparathae
Hack-Craft reimagines the hackathon portal as an immersive 3D Minecraft biome. Solving low engagement through gamified UX, it features a custom Villager-in-a-Boat theme engine, glassmorphic UI, and 3D parallax landscapes. From "Mining Shaft" timelines to "Loot" prize grids, it turns registration into a quest. Responsive, bold, and RTX-on.
